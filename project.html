Innovating agricultural automation with robotics and IoT

Abstract

Coconut harvesting is a labour-intensive and hazardous task, often requiring skilled climbers to scale tall trees. To address this challenge, this project focuses on the design and implementation of an unmanned remote-controlled coconut tree climbing and harvesting robot. The system consists of a mechanized climbing mechanism, a cutting tool, and a wireless control interface, enabling safe and efficient harvesting.
